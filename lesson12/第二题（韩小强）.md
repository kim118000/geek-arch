2 、分析如下HiveQL，生成的MapReduce执行程序，map函数输入是什么？输出是什么，
reduce函数输入是什么？输出是什么？

INSERT OVERWRITE TABLE pv_users
SELECT pv.pageid, u.age
FROM page_view pv
JOIN user u
ON (pv.userid = u.userid);



map函数输入的是

| userid | pageid |
| ------ | ------ |
| 111    | 1      |
| 111    | 2      |
| 222    | 1      |

| userid | age  |
| ------ | ---- |
| 111    | 25   |
| 222    | 32   |

map输出是

| key  | value  |
| ---- | ------ |
| 111  | <1,1>  |
| 111  | <1,2>  |
| 222  | <1,1>  |
|      |        |
| 111  | <2,25> |
| 222  | <2,32> |

reduce函数输入

| key  | value  |
| ---- | ------ |
| 111  | <1,1>  |
| 111  | <1,2>  |
| 111  | <2,25> |
|      |        |
| 222  | <1,1>  |
| 222  | <2,32> |
|      |        |

reduce输出是

| pageid | age  |
| ------ | ---- |
| 1      | 25   |
| 2      | 25   |
|        |      |
| 1      | 32   |
|        |      |
|        |      |
|        |      |
